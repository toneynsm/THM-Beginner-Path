## Linux Shell
- Mainly just noting Bash (Born again shell) right now
- Scripting: when scripting in Linux, you need to specify the interpreter with which the set of commands (script) is to actually be RUN with. This is known as the **interpreter directive**, and looks like "#!" (Ex. #1/bin/bash = specifying you will use BASH to run the upcoming commands, as opposed to other shells)
